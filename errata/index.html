Errata for First Edition

  • Page 53: In the proof of $\text{trace}(AB)=\text{trace}(BA)$, if $A$ is $d\times N$ and $B$ is $N\times d$, the display should be $$\sum_{i=1}^d (AB)_{ii}=\sum_{i=1}^d \sum_{k=1}^N a_{ik}b_{ki}= \sum_{k=1}^N \sum_{i=1}^d b_{ki}a_{ik}=\sum_{k=1}^N (BA)_{kk}.$$
  • Page 66: The second display should be $x^+= C^+f = \dfrac1{10}(-8,-5,-2,1,4)$.
  • Page 74: The first and second displays should be $$10f = -8a-5b-2c +d+4e,\qquad a = a+0f,\quad b = 2a+f,\quad c= 3a+2f,\quad d= 4a+3f,\quad e=5a+4f.$$
  • Page 208: In Archimedes' axiom, "than that" should be "then that".
  • Page 259: In Exercise 4.5.13, for the converse, assume also $f(x)$ is convex.
  • Page 377: In the footnote, the $p$-value should be $Prob(|T| > 1)$, and the speed of light is $\sqrt{d}$.
  • Page 396: Weights $m$, $w$ are row vectors and samples $x$, $\tilde x$ are column vectors, so the display should be $$w=(m,b),\qquad \tilde x=\begin{pmatrix} x\\1\end{pmatrix}.$$
  • Page 410: Equation (7.2.13) corresponds to edge $(j,i)$, not $(i,j)$.
  • Page 411: The weight matrix $M$ and the bias vector $b$ are $$M= \begin{pmatrix} m_1\\ m_2\\ m_3\end{pmatrix},\qquad b=\begin{pmatrix} b_1\\ b_2\\ b_3\end{pmatrix},\qquad W=(M,b),$$ so $M$ is $3\times4$, $b$ is a column vector in ${\bf R}^3$, and the augmented weight matrix $W$ is $3\times5$.
  • Page 457: The last sentence before the code should be "a matrix with a single column".
  • Page 520: At bottom, the last sentence should be "the hermitian product $P\overline{P'}$ of $P$ and $P'$".
  • Page 543: The definition of closed should be "A set $K$ is closed if $x_n$ in $K$ and $x_n\to x$ implies $x$ is in $K$".
